## Installing Nerd Fonts
- Visit [Nerd Fonts](https://www.nerdfonts.com/font-downloads) and download the font **Hurmit Nerd Font** (Reserved Font Name: Hermit).
- Unzip the downloaded .zip file and install the fonts.
\
For installing, you can copy and paste all the fonts to `/usr/local/share/fonts` and reboot (or manually rebuild the font cache with `sudo fc-cache -f -v`). This is make the fonts available system-wide (for all users of your computer).

## Terminal Configuration 
- Open GNOME terminal.
- Hamburger Icon > **Preference**.
- Create a new profile clicking on the **plus (+)** icon on **Profile**.
- Under **Text** tab, check **Custom font** and select **Hurmit Nerd Font Mono** and size **13**.
- Under **Colors** tab, from **Text and Background Color** uncheck **Use colors from system theme**. From **Built-in schemes** dropdown, select **Custom** and set the **text** color as **#4AF626** and **background** color as **#1A1A1B**.
- Finally, from the sidebar click on the dropdown arrow that appears after the profile name you just created and click **Set as default**. Close and open terminal again to enjoy the new look.
